# UX Foundations Pro Quiz

## Project Overview
**UX Foundations Pro Quiz** is a high-fidelity UX Foundations Pro quiz application with modern design principles and smooth interactions that tests design knowledge.

## Key Features Implemented:
- **Educational Content:**
50 questions covering UX Laws (Fitts's, Hick's, Jakob's), UI principles (visual hierarchy, WCAG 2.2, whitespace), design process, and micro-interactions
Each question includes detailed explanations based on real design theory

- **UX/UI Excellence:**
Dynamic progress bar with percentage indicator
Immediate visual feedback (green for correct, red for incorrect)
Smooth slide-in animations for questions
Mobile-responsive design using Tailwind CSS

- **Scoring System:**
Real-time score tracking

- **Final results screen with percentage and designer level classification:**
90%+: UX Master
80-89%: Senior Designer
70-79%: Mid-Level Designer
60-69%: Junior Designer
Below 60%: Design Novice

- **Visual Design:**
Clean, minimalist SaaS aesthetic
Professional color palette (Deep Slate #1e293b, Indigo #4f46e5)
Subtle shadows, rounded corners, and micro-interactions
System font stack for crisp typography

The quiz provides an engaging learning experience while testing knowledge of modern UX/UI principles. Try it out!
